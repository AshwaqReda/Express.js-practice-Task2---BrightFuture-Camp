This project is a simple backend application built using Node.js and Express.js for practicing the basics of backend development.

#It includes:

- Creating an Express server
- Handling GET and POST requests
- Working with JSON data
- Returning proper HTTP status codes
- Using middleware like express.json()
- Using morgan for request logging

#Technologies Used
- Node.js
- Express.js
- Morgan

#This project was created as part of a backend practice task.
